# 🍽️ DineKart – Food Ordering Platform

🚀 A backend-focused food ordering system designed to manage orders, users, and restaurant operations efficiently with real-time capabilities.

---

## ✨ Features

* 🛒 Order management system
* ⚡ Real-time order updates using Socket.IO
* 🏪 Multi-vendor architecture for independent restaurant management
* 🔐 Role-based authentication (Admin / Vendor / User)
* 📊 Optimized APIs and database queries for better performance

---

## 🛠️ Tech Stack

**Backend**

* 🟢 Node.js
* 🚀 Express.js

**Database**

* 🍃 MongoDB

**Other Tools**

* 🔌 Socket.IO
* 🧪 Postman
* 🗂️ Git & GitHub

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash id="p6p4wq"
git clone https://github.com/shivansh1609/DineKart.git
cd DineKart
```

### 2️⃣ Setup Backend

```bash id="u5y8g1"
cd backend
npm install
npm start
```

### 3️⃣ Setup Frontend

```bash id="3p4l8o"
cd frontend
npm install
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in backend folder:

```env id="j9xv8m"
PORT=5000
MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret_key
```
